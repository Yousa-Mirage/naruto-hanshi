# 《火影忍者Online》寒食春宴计算器

> 适用于 2026.4.3 ~ 2026.4.9 的寒食春宴活动

《火影忍者Online》寒食春宴活动食物计算器，最大化获得的星辉券数量。

输入当前拥有的食材数量，页面会计算在现有约束下的最优制作方案，并展示：

- 可制作的食物数量
- 每种食物的制作份数
- 各食材的消耗与剩余

项目当前按“每份食物等于 10 个星辉券”的前提，等价地最大化食物总数。

## 当前食谱

参与求解的食谱：

- 寒食粥：水×4，米×1
- 醴酪：糖×2，麦仁×3，水×3
- 子福：枣×3，水×2，面粉×3
- 寒食饼：面粉×4，水×1
- 寒食面：米×1，水×1，面粉×3
- 寒食浆：糖×1，水×2，面粉×2
- 饧：糖×6，水×4

已排除的食谱：

- 清明馃：需要额外食材“艾草”，且不产出星辉券
- 青精饭：需要额外食材“乌饭树叶”，且不产出星辉券
- 子推燕：被寒食浆严格支配，不会出现在最优解中

## 算法

项目使用整数线性规划求解：

- 决策变量：每种食物制作多少份
- 目标函数：最大化食物总数
- 约束条件：每种食材的总消耗不能超过用户输入库存
- 整数约束：每种食物数量必须是非负整数

求解后会进一步计算：

- 各食谱的实际制作数量
- 每种食材的总消耗
- 每种食材的剩余数量

使用 [`YALPS`](https://github.com/IanManske/YALPS) 库在前端进行规划求解计算。

## 本地运行

首先需要安装 npm，然后运行以下命令启动本地服务：

```bash
git clone https://github.com/Yousa-Mirage/naruto-hanshi.git
cd naruto-hanshi
npm install
npm run dev
```
